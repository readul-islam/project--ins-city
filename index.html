LocalStorage Vs SessionStorage

The storage object save data in web browsers The localstorage shared data between all tabs and windows from the same origin. Localstorage data will not be expired when the browser is closed. But, The sessionStorage opposite the localstorage. Session storage lost data when the browser tab is closed. This is the main difference between local storage with sessionStorage.
